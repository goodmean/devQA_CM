# 프로젝트 주제 : 개발자 Q&A 커뮤니티

# 시작 일자 : 2022.06.19 ~
- [git](https://github.com/goodmean/devQA_CM)

# 기술스택
- 백엔드 : Java / spring Boot
- 프론트 : thymeleaf
- 빌드툴 : gradle
- ORM(Object Relational Mapping) : Spring data JPA
- Spring Security
- DB : MariaDB
- 빌드 : Jar
- IDE : [intelliJ](https://www.jetbrains.com/ko-kr/idea/download/#section=windows)

# 기능
- User (회원)
	- join(회원가입)
    - login(로그인)
    - logout(로그아웃)

- Question(질문)
	- list(리스트)
    - detail(상세내용)
	- create(등록)
    - modify(수정)
    - delete(삭제)
    - search(검색)
    
- Answer (답변)
	- create(등록)
    - modify(수정)
    - delete(삭제)
    
- 리스팅/추천/검색
